# Phase 9-3 개발 노트 — 학습 효과 정교화 (비개발자용)

3차 코드리뷰("구현 품질보다 학습 효과를 더 끌어올릴 여지") **6건 전부(9-3a~f)**를 반영했습니다. 가벼운 로직·데이터·UX(9-3a/b/c/e)를 먼저 반영하고, 이어서 스키마 변경(9-3d)·콘텐츠 구성(9-3f)이 동반되는 항목까지 완료했습니다.

## 9-3a — 이전 레벨 복습이 '오늘의 학습'에 다시 끼어들게

- **무엇이 문제였나:** 레벨이 올라가면, 예전 레벨에서 "복습 필요"로 표시된 단어들이 데이터엔 남아 있어도 **오늘의 10단어에는 거의 안 나왔습니다.** (오답 노트에서만 볼 수 있었음)
- **무엇이 바뀌나:** 이제 복습할 때가 된 단어는 **레벨과 상관없이** 오늘의 세션에 자연스럽게 섞입니다. 오래 안 본 단어가 잊히기 전에 다시 등장 → 기억 유지율이 올라갑니다.
- 단, **카테고리 학습**(예: 여행)은 주제 흐름을 지키기 위해 그 범위 안의 복습만 섞습니다.

## 9-3b — 마이크가 자꾸 못 알아들을 때 지치지 않게

- **무엇이 바뀌나:** 따라 말하기에서 음성이 **두 번 연속 인식 실패**하면, 점수를 깎거나 계속 다시 시키지 않고 "🙂 인식이 잘 안 되는 환경 같아요. 오늘은 듣고 넘어가도 좋아요"라고 안내하며 부드럽게 넘어갑니다.
- 이건 **사용자 발음 탓이 아니라 환경(마이크·소음) 문제**로 처리하므로, 발음 점수나 약점 단어에 전혀 불이익이 없습니다.

## 9-3c — '겨우 맞힌 것'과 '바로 맞힌 것'을 구분

- **무엇이 문제였나:** 여러 번 틀리고 힌트를 본 뒤 맞혀도, '뜻 회상' 점수가 **바로 맞힌 것과 똑같이 만점**이었습니다.
- **무엇이 바뀌나:** 이제 힌트를 본 횟수와 답하는 데 걸린 시간을 기록해서, **힌트로 겨우 떠올린 단어는 회상 점수를 낮게** 매깁니다. (힌트 1개당 감점, 너무 오래 걸리면 약간 감점) 한 번에 바로 맞히면 그대로 만점입니다.
- 덕분에 "사실은 아직 헷갈리는 단어"가 통계에 더 정확히 잡혀, 복습·레벨 판단이 정교해집니다.

## 9-3e — 틀린 단어, 바로 넘기지 않고 한 번 정확히 입력

- **무엇이 문제였나:** 타이핑에서 **하트가 다 떨어지면(틀리면)** 곧장 따라 말하기 단계로 넘어가, 정작 정답을 또렷이 익힐 기회가 약했습니다.
- **무엇이 바뀌나:** 이제 하트가 소진되면 **정답을 보여주고, 그 단어를 한 번 그대로 따라 입력**하게 한 뒤 다음 단계로 넘어갑니다.
  - 이 복구 단계에서는 **하트가 더 깎이거나 틀렸다고 혼나지 않습니다.** (잘못 입력하면 입력창이 살짝 흔들릴 뿐)
  - 급하면 **[건너뛰기]**로 바로 넘어갈 수 있어, 학습이 막히지 않습니다.
- "틀린 단어를 손으로 한 번 정확히 써보는" 작은 과정이 추가돼, **오답이 기억에 더 잘 남습니다.** (이 단어는 자동으로 '아직 약한 단어'로 기록 — 9-3c와 연동)

## 9-3d — '철자 졸업'과 '발음 졸업'을 따로 관리

- **무엇이 문제였나:** 회화앱인데 졸업 판단이 하나의 점수에 묶여 있었습니다. 마이크가 없는 환경(typingOnly)에서는 발음 연습 자체가 불가능한데도요.
- **무엇이 바뀌나:** 단어의 **타이핑(철자) 숙련**과 **발음 숙련**을 별도로 추적합니다.
  - 타이핑을 충분히 익히면(첫 시도 정답 3회) 그 단어는 **복습 더미에서 졸업**합니다. 마이크가 없어도 막히지 않습니다.
  - 단, 발음이 아직이면 **나중에 마이크가 되는 환경에서 "말하기 확인"으로 아주 가끔(세션당 최대 1개) 다시 등장**해, 한 번 소리 내어 확인하게 합니다. (틀려도 벌점 없음)
- 즉 "손으로는 익혔지만 입으로는 아직"인 단어를 놓치지 않으면서도, 학습이 환경 때문에 막히지 않습니다.
- 이 기능은 데이터 한 칸(`pron_pass_count`)이 추가돼 **DB 마이그레이션 1회**(`v10`)가 필요합니다. 적용 전이라도 자동으로 떼고 저장하므로 학습은 멈추지 않습니다.

## 9-3f — 카테고리 학습을 '하나의 상황 이야기'로

- **무엇이 문제였나:** "여행" 카테고리를 골라도 단어 10개가 **서로 무관하게 흩어진 묶음**에 가까웠습니다.
- **무엇이 바뀌나:** 이제 10문항이 **하나의 상황 흐름**을 이루도록 묶습니다. 예: 여행 → *공항 → 이동 → 길 찾기 → 숙소* 처럼 인접한 장면의 표현들이 한 세트로 나옵니다.
  - 그중에서도 **아직 안 배웠거나 복습이 필요한 표현이 가장 많이 모인 구간**을 골라줘서, 흐름도 자연스럽고 학습 효율도 높습니다.
- 회화로 옮겨가는 전이(실제 상황에서 말이 나오는 것)가 더 좋아집니다. **새 콘텐츠 제작 없이** 기존 단어 태그(여행·공항·길찾기 등)를 활용했습니다.
