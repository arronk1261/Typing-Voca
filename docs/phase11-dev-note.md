# Phase 11 개발 노트 — 로그아웃 상태 서비스 소개(랜딩) 페이지

> 비개발자(기획자)용 브리핑. "사용자가 무엇을 보고, 어떤 동작을 하고, 화면이 어떻게 반응하는지" 관점.

## 한 줄 요약

로그인하지 않은 사람에게 곧장 "구글로 로그인" 버튼만 던지던 화면을, **무엇을·왜·어떻게 배우는 앱인지 보여주는 매력적인 소개 페이지(`/welcome`)** 로 바꿨습니다. 학습 원리(직접 치고 말하는 능동 학습 + 망각 곡선에 맞춘 복습)·레벨 구성·재미 요소·학습 대상을 스크롤로 풀어 보여주고, 화면 곳곳에서 한 번에 가입하도록 유도해 **이용 전환률**을 끌어올립니다.

## 무엇이 달라졌나

- **이전:** 로그인 안 한 사용자가 앱에 들어오면 → 설명 없는 로그인 카드만 노출.
- **이후:** 로그인 안 한 사용자가 앱에 들어오면 → **서비스 소개 랜딩(`/welcome`)** 으로 자동 안내 → 충분히 둘러보고 마음먹은 뒤 [Google로 시작하기].
- 이미 로그인한 사용자나 게스트 모드는 **이전 그대로** (랜딩을 거치지 않고 바로 대시보드).

## 사용자가 보게 되는 것 (위에서 아래로 스크롤)

1. **첫인상(Hero)** — "손끝으로 치고, 입으로 말하며 외우는 영어 회화" 한 줄 가치 + 실제 학습 화면을 본뜬 미니 미리보기(빈칸 `l a t e`가 한 글자씩 채워지고 ★★★). 바로 아래 큰 [Google로 시작하기].
2. **학습 루프 3단계** — ①한글 뜻 떠올리기 → ②빈칸 직접 타이핑 → ③문장 따라 말하기. "보고 넘기는 게 아니라 직접 꺼내 쓴다"는 차별점을 카드로.
3. **망각 곡선 설계(과학적 근거)** — 그래프 한 장으로 핵심을 전달: 그냥 외우면(빨강) 기억이 며칠 만에 급락하지만, 잊을 때쯤 다시 묻는 복습(파랑)은 톱니처럼 높게 유지된다. 아래에 "SM-2 간격 반복 / 복습 우선 출제 / 두 번 통과해야 졸업" 설명 카드.
4. **레벨 구성** — Lv.1~3 + "잘하면 올리고 버거우면 낮추는" 적응형, 신규 70%·복습 30% 균형, 12개 생활 카테고리 칩.
5. **재미 요소** — 스트릭🔥·배지 26종·학습 3링·정답 컨페티·스트릭 동결권·위클리 챌린지를 2열 갤러리로.
6. **학습 대상 + 안심 배지** — "여행·직장·일상 회화가 목표인 성인" 등 대상 4가지 + 끊기지 않는 학습·목소리 미저장·기기 간 이어가기.
7. **마지막 권유(Final CTA)** — "오늘 10단어부터 시작해 볼까요?" + 큰 버튼.

- 상단에는 **고정 헤더**(로고·다크모드 토글·로그인), 하단에는 **항상 떠 있는 [무료로 시작하기] 버튼**이 따라다녀 어느 위치에서든 한 번에 가입 가능합니다.
- 모든 섹션은 스크롤하면 **부드럽게 떠오르는 등장 효과**가 있고, 동작 최소화를 켠 기기에서는 효과가 자동으로 꺼집니다.
- 라이트/다크 모드 모두에서 점검 완료, 모바일 세로(390px) 기준으로 설계했습니다.

## 사용자가 하는 동작 / 화면 반응

- 아무 CTA(상단 로그인·중간 버튼·하단 고정 버튼)나 누르면 → 곧바로 구글 로그인 창. 중간에 거치는 화면이 없어 마찰이 적습니다.
- 로그인을 마치면 기존과 동일하게 대시보드(또는 최초라면 레벨 테스트)로 들어갑니다.

## 안심 포인트 (기획 관점)

- **학습에는 영향 없음:** DB나 학습 로직은 전혀 건드리지 않았습니다(순수 소개 화면 추가). 운영 반영(마이그레이션·시드)도 필요 없습니다.
- **기존 흐름 보존:** 로그인 사용자·게스트 모드는 그대로. 직접 `/login`으로 들어가는 길도 남겨 뒀습니다.
- 망각 곡선 그래프와 설명 문구는 모두 실제 앱이 쓰는 방식(SM-2 간격 반복·신규 70%/복습 30%·2회 통과 졸업)에 근거해, 과장 없이 사실대로 설명합니다.

- 라우트: `src/app/welcome/page.tsx`(서버 컴포넌트, OG 메타) → `WelcomeView`(클라이언트, 로그인/미설정 시 `/` 리다이렉트) → `LandingPage`(헤더·sticky CTA).
- 섹션: `src/components/landing/` — `Reveal`·`GoogleCta`·`SectionHeading`·`Hero`·`LoopSteps`·`ForgettingCurve`(recharts)·`LevelSection`·`FunGallery`·`AudienceSection`·`FinalCta`·`LandingPage`·`WelcomeView`.
- 가드 연결: `AuthGuard`·`Dashboard` 비로그인 분기 → `/welcome` 리다이렉트(`GoogleLoginCard` 직노출 제거).

- QA 미리보기: `.claude/launch.json`의 `dev-loggedout`(포트 3200) — 로그인-퍼스트 + 무세션 상태를 흉내 내 `/welcome`을 미리보기 위한 전용 구성.
